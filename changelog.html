Payment-Only Redesign Website Refresh

released

Public site and document surfaces were aligned to the current Atho payment-only protocol baseline on April 18, 2026.

Scope includes homepage simplification, documentation hub rewrite, wallet and wallet hierarchy refresh, retirement of the old inflation/rewards landing pages, and bundle-ready IONOS output sync.

  • Removed outdated protocol and payout language from the public site so the live baseline stays focused on payments, privacy, and current constants.
  • Updated public constants to 6 confirmations, 100 ATHO starting reward, 720 blocks per day, and 1,000,000,000 ATHO max supply.
  • Reframed Atho around its current product fit: proof-of-work payments with public and private transaction support.
  • Rebuilt the wallet hierarchy page from the latest docs and added click-to-expand diagram viewing.
  • Prepared the site for refreshed docs catalog generation and IONOS bundle upload.
