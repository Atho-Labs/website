Consensus Update: Pre-tail Base Alignment

released

This release aligns economics accounting so the bootstrap allocation is included in the fixed pre-tail target exactly as intended by policy.

Published March 2026. Scope includes consensus constants, emission simulation references, and synchronized documentation updates.

Previous interpretation

Pre-tail accounting effectively treated bootstrap outside the 30M target.

  • Path: 30,000,000 subsidy + 50,000 bootstrap.
  • Effective total: 30,050,000 before tail.
  • Tail switch: block 21,102,000.

Current active policy

Bootstrap is now included inside the fixed pre-tail total by design.

  • Path: 29,950,000 subsidy + 50,000 bootstrap.
  • Pre-tail total: exactly 30,000,000.
  • Tail switch: block 20,942,000.
Previous Consensus Alignment Current

Post-tail issuance remains 0.25 ATHO per block (65,700 ATHO/year), and this update moves the tail switch earlier by 160,000 blocks to keep economics and documentation consistent.
