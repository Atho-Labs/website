Testnet Quickstart

Build and run the Atho testnet client.

This is the public testnet operator path. Pull the repository, build the binaries, run the launcher, create a wallet, sync to the network, and start mining when the client is fully caught up.

Open Testnet Repo Join Testnet Email for Coins
Step 1

Install prerequisites

Install Rust and Python on your machine before building the testnet client.

Step 2

Build the binaries

Open a terminal in the repository root and run cargo build or cargo build --release.

Step 3

Run the launcher

Start the client with python testnet.py. If it does not open on the first run, run it again.

Published

Create wallet, sync, and mine

Create a wallet in the client, wait until the node is synced, and then start mining or send testnet transactions.
