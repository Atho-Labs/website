Public Testnet Status

The public path is the Atho testnet.

The public website now points directly at the testnet flow: build the code, run the client, create a wallet, sync, mine, and request testnet coins by email.

Join Testnet Open Testnet Repo

Current public flow

  • Clone or pull the Atho-Testnet repository.
  • Build with cargo build or cargo build --release.
  • Run python testnet.py to open the client.
  • Create a wallet, sync to the network, and start mining when fully synced.

Testnet coin flow

  • Create a testnet wallet and copy a testnet address.
  • Take a screenshot of the synced client.
  • Email the screenshot and the address to genull@proton.me.
  • Coins will be sent over for mining and transaction testing.

Atho is building quantum-secure payment infrastructure with simple rules, durable chain truth, and a long-term protocol posture.
